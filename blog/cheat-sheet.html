Ruby Enumerable methods and JavaScript equivalents

This cheat sheet summarizes the most commonly used Ruby enumerable methods and their Javascript equivalents, and provides example code for each.

  • #each and .forEach()
  • #map and .map()
  • #select and .filter()
  • #inject and .reduce()
  • #all? and .every()
  • #any? and .some()

#each and .forEach()

#each calls the given block once for each element, passing that element in as parameter. The JavaScript equivalent is .forEach().

#map and .map()

#map creates a new array using the values returned by calling the block for each element. The JavaScript equivalent is .map().

#select and .filter()

#select (or #find_all) returns a new array of all values that pass as true in the given block. The JavaScript equivalent is .filter().

#inject and .reduce()

#inject (or #reduce) combines all elements in an array by passing each element into a block. The "accumulator" is the first parameter specified within the pipes of the code block and can optionally be given a starting value by including an argument with the #inject method. The iterated variable is the second parameter included in the code block. The JavaScript equivalent is .reduce(). Note that the optional starting value is placed as the second parameter of the .reduce() method.

#all? and .every()

#all passes each element to the block, and returns "true" if the block never returns false or nil. The JavaScript equivalent is .every().

#any? and .some()

#any passes each element to the block, and returns "true" if any element evaluates as true. The JavaScript equivalent is .some().
